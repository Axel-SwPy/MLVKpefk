Завдання №1
Створити Web-сторінку, що містить вірш Василя Симоненка "Ти знаєш, що ти людина". Для відображення назви вірша "Ти знаєш, що ти люжина..." використайте заголовок першого рівня (тег h1). Вирівнювання стовпчиків вірша та останнього абзацу проведіть у два прийоми:

  1. Все вирівняйте по центру. зроблену роботу покажіть вчителю.
  2. Пеший стовбчик вірша вирівняйте по лівому краю, другий по центру, третій по равому краю. Останній абзц вирівняйте по ширині (align='justify').

На усій Web-сторінці використайте шрифт темно-синього кольору (navy) та за назвою Consolas.
Виконану роботу покажіть вчителю.
Змініть колір назви вірша на червоний.
Змініть колір другого стовбчика вірша на зелений, а шрифт на Impact.
Виконану роботу покажіть вчителю.
Текст вірша і останнього абзацу знайдете у папці HTML-матеріали/До_теми_02/simonenko
Завдання №2
Створити Web-сторінку, що містить фрагмент оповідання Льюіса Керрола "Аліса в країні чудес".

  • Перший рядок сторінки - заголовок першого рівня кольору orange.
  • Другий і третій рядки - заголовок третього рівня (між ними не повинно бути порожнього рядка).
  • Наступні абзаци вирівнюються по ширині сторінки зі шрифтом Comic Sans MS.

Виконану роботу покажіть вчителю.
Змініть колір трьох абзаців тексту оповідання:
  • перший абзац - червоний (red);
  • другий абзац - зелений (green);
  • третій абзац - синій (blue);

Виконану роботу покажіть вчителю.
Текст знайдете у папці HTML-матеріали/До_теми_02/alisa_1
